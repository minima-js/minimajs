# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-03

### Added

- Initial release of @minimajs/openapi
- OpenAPI 3.1 specification generation
- Zod schema to OpenAPI schema conversion
- Route metadata collection via `doc()` helper
- Support for query parameters, headers, path parameters, and request bodies
- Response schema definitions with status codes
- Tags and operation metadata
- Security scheme definitions
- Server and contact information
- Example file demonstrating usage
- Comprehensive README with usage examples
